# Viral Pattern Notes

## Common Hook Types
- `question`: Start with a sharp question.
- `contrarian`: Challenge common belief.
- `risk-alert`: Highlight danger/cost of inaction.
- `result-first`: Start with concrete result.
- `list-promise`: Promise structured list of takeaways.

## High-Performance Body Patterns
- `problem -> method -> proof -> action`
- `context -> mistakes -> fixes -> checklist`
- `claim -> examples -> boundaries -> CTA`

## CTA Types
- Ask for experience sharing.
- Ask for comment-based contribution.
- Ask user to save/bookmark for later.

## Anti-Copy Rule
- Keep structure, not wording.
- Rewrite examples in user's domain.
- Introduce at least one original perspective.
